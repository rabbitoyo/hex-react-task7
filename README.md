## React 作品實戰班 Vite & React - Redux 與 Redux Toolkit

此專案為六角學院 React 作品實戰班 Vite & React 作業

- [GitHub Page Demo](https://rabbitoyo.github.io/hex-react-task7/)

### 🛠 使用技術

- Vite
- Bootstrap 5
- SCSS
- React

### 💻 安裝執行

- 將專案 clone 到本地端

```bash
git clone https://github.com/rabbitoyo/hex-react-task7.git
```

- 安裝執行

```bash
npm install				// 安裝
npm run dev				// 執行
```

- 瀏覽器訪問

```bash
http://localhost:5173/
```
